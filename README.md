# Bunker-Anomaly-Game
The Bunker Anomaly Game is my Stanford Code in Place 2026 Final Project.
This game is inspired by the populr game The Exit 8. The player moves in a particular direction based on the presence of anomalies.
The game is set up in a bunkler environment. The player is set up to be a silly looking charachter that got lost and ended up there. Now, the player is trying to escape the bunker, by identifying the anomalies.
Anomalies of various difficulty levels show up and the player is suppposed to choose its direction accordingly.
To win the game, the player must choose correctly in all five levels. 
# Features
- Start Screen
- Win and Lose screens
- Restart Option
- Instructions and Score display
- Background Music
- Various fonts and colors
- Images for player and backgrounds
- 7 different anomalies
